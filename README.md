# dotfiles for cygwin by masaakif.

This is dotfiles for cygwin environment.

# How to use.

1. Install cygwin with git.
1. Start cygwin bash terminal.
1. Enter command as below to install dotfiles.

```
$ bash -c "$(curl -L ***)"
```
